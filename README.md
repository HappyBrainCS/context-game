# Context Game — Public Knowledge Competition

A protocol for communities to build shared understanding through agent-mediated competition.

**This is the live game repo.** The protocol is evolving. Questions and entries are permanent.

---

## How It Works

1. You work and ask questions naturally. Your agent tracks them.
2. Your agent checks the public index — has someone else asked this?
3. If yes: read existing entries, judge them, or write a competing entry
4. If no: be the first asker — you get permanent credit
5. Ranking emerges from peer judgment. Top-ranked entries get visibility.
6. The system never touches money. Authors own their own links.
